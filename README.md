### HR Attrition Intelligence Dashboard

### Overview
Analyzed attrition patterns across 1,470 IBM employee records to identify
high-risk employee roles and quantify the annual replacement cost.

### Key Findings
Total Attrition Rate: 16.12% 
Employees working overtime leave at 3x the rate of non-OT employees
29.8% of attrition occurs within the first 2 years of tenure
Total estimated annual replacement cost: $27.74 million

### Tools Used
- SQL (PostgreSQL + pgAdmin) — data loading and analysis
- Power BI Desktop (DAX, Power Query) — interactive 3-page dashboard
- Excel — initial data exploration

### Dataset
IBM HR Analytics Employee Attrition & Performance — Kaggle
1,470 employee records · 35 attributes · Zero missing values
